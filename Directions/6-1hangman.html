Requirements

In this assignment, you will recreate hangman. If you are unfamiliar with hangman, you can view the rules here.

This will be a singleplayer version of the game. To generate the words, we will use the random_words library to generate our words. (Note: if you find a better library please let me know!)

I have provided to 2 methods to help you out.

  • generate_word(): Using the random word library, this generates a word for the user to guess
  • generate_blank():This converts the word to be guessed into blank so you can print the length to the screen

You should keep track of the letters guessed by the player and print out every letter they have guessed after a guess.

Notes

  • Strings are immutable in Python, which means once they are created they cannot be changed.
