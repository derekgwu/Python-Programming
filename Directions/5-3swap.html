Chapter 5.3 Exercise: Swap Two Numbers

Goals

  • Loops
  • Arrays
  • Translating Requirements to Code

Requirements

Write a method that when given an array and two numbers, it swaps the two numbers' position in the array and returns the array. For example, if the array was [1,2,3,4] and the two numbers were 1 and 3, you would return [3,2,1,4]

Hints

You'll need a holder variable to remember which index the first number to swap was at. Then, when you iterate over the second number, you can easily swap the two.

Additionally, the swapping process will require a temporary variable.
